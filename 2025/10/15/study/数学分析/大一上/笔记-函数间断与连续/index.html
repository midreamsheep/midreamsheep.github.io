连续 ⇔ 左连续 + 右连续

有限次四则运算不改变连续性 ⇔ C为一个空间

区间连续

f在(a, b)中的每一个点均连续，那么称f在(a, b)上连续，记为f ∈ C(a, b)

f在[a, b]连续 ⇔ f ∈ C(a, b)且f在a右连续且f在b左连续

复合函数的连续性

f在x0连续，g在f(x0)连续 ⇒ g(f(x))在x0连续

∀ϵ > 0, ∃δ > 0, s.t.对∀|y − f(x0)| < δ, 有|g(y) − g(f(x0))| < ϵ

对上述确定δ，∃δ2 > 0, s.t.∀|x − x0| < δ2, 有|f(x) − f(x0)| < δ, 则有|g(f(x)) − g(f(x0))| < ϵ

类比较原理

f, g在x0连续，f(x0) < g(x0)（不能有等于），那么∃U(x0), 对∀x ∈ U(x0), s.t.f(x) < g(x)

对x ∈ Ů(x0), f(x) ≤ g(x)（或 < ） ⇒ f(x0) ≤ g(x0)

介值定理
