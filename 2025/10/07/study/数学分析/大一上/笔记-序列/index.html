各种序列的定义及其描述式

有界无界

有界：∃M > 0, s.t.∀n ∈ ℕ, 有|vn| < M

无界：∀M > 0, ∃n ∈ ℕ, s.t.|vn| > M

发散

发散：不收敛序列称为发散序列–>见下方收敛序列

发散于正无穷：∀M > 0, ∃N ∈ ℕ, s.t.∀n ≥ N, 满足vn < M

不发散于正无穷：∃M > 0, s.t.∀N ∈ ℕ, ∃n ≥ N, 满足vn < M

发散于负无穷，不发散于负无穷：略

无穷大小

无穷大序列：∀M > 0, ∃N ∈ ℕ, s.t.∀n ≥ N, 有|vn| > M

非无穷大序列：∃M > 0, s.t.∀N ∈ ℕ, ∃n ≥ N, 满足|vn| < M

发散于正无穷|发散于负无穷|发散于正负无穷 即为无穷大序列

不发散于正无穷&不发散于负无穷 即为非无穷大序列

无穷小序列：∀ϵ > 0, ∃N ∈ ℕ, s.t.∀n > N, 有|vn| < ϵ

无穷小序列一定有界

非无穷小序列：∃ϵ > 0, s.t.∀n ∈ ℕ, ∃n > N, 有|vn| > ϵ

重要结论（收敛性质）

极限存在必唯一

证明：

取N̂ = max{N1, N2}即可证明矛盾

收敛必有界

取ϵ = 1, ∃N ∈ ℕ, s.t.∀n ≥ N, 有|xn − a| < 1

即|xn| < 1 + |an|,取M = max{|x1|,|x2|,...|xN − 1|,1 + |a|}

则∀n ∈ N, |xn| ≤ M, 即序列有界

单调有界序列必收敛

因为xn有界，则xn有上确界，故取a = supx1, ..., xn

因此∀ϵ > 0, ∃N ∈ ℕ, s.t.a − ϵ < xn ≤ a

又因为{xn}严格单增，故对任意n ≥ N, 有a − ϵ < xn < a + ϵ

得证

B-W定理

若a0 ≤ xn ≤ b0很成立，则一定存在一个收敛子序列{yn}

证明：通过二分法构造区间套[an, bn]与{yn}且yn ∈ [an, bn]

柯西收敛原理（Cauchy）

{xn}收敛 ⇔ ∀ϵ > 0, ∃N ∈ ℕ, s.t.∀n, m ≥ N, 有|xn − xm| < ϵ

{xn}不收敛 ⇔ ∃ϵ0 > 0, 对∀N ∈ ℕ, ∃n ≥ N, ∃p ∈ ℕ, s.t.|xn + p − xn| > ϵ0

收敛->柯西

因为{xn}收敛，故∀ϵ > 0, ∃N ∈ ℕ, s.t.∀n ≥ N, 有|xn − a| < ϵ

得证

柯西->收敛

(1)证明{xn}有界

取ϵ = 1, 则∃N ∈ ℕ，对∀m, n > N, 有|xn − xm| < 1 ⇒ |xn − xN| < 1 ⇒ |xn| < |xN|+1

取M = max{|x1|,|x2|,..., |xN − 1|,|xn + 1|}, |xn| ≤ M

(2){xn}有界 ⇒ ∃收敛子序列{xnk}, 令其极限为a

(3)证明{xn}极限为a

则取N = max{N̂, Ñ}, 则|xn − a| ≤ |xn − xnk|+|xnk − a| ≤ ϵ

得证
