函数间断点

在x = 0处有振荡间断点

在x = 0出有可去间断点

在x = 0处连续，在x ≠ 0处不连续

处处不连续

(2)∀x0 = 0或者x0 ∈ ℝ − ℚ, f(x0) = 0

函数有界

取ϵ0 = 1, ∃δ > 0, s.t.∀0 < x < δ, 有|f(x) − 1| < ϵ0, 即f(x)在(0, δ)有界

有易得f在[δ, 1]上连续，则在其上有界，则在[δ, 1)上有界

因此f在(0, δ) ∪ [δ, 1) = (0, 1)上有界

一致连续

一致连续函数：f(x) = x2在(0, a)上，其中a ∈ (0, +∞)

不一致连续函数：f(x) = x2在(0, +∞)上
