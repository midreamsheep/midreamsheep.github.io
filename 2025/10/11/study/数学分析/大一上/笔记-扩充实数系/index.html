定义

将集合ℝ ∪ {+∞, −∞}称为扩充实数系R̄

注：这只是一个命名，并不能真的把+∞, −∞当成数来看待

极限和确界记号

  1. $序列\{x_n\}发散到正无穷，记为\lim\limits_{n \to +\infty}x_n = +\infty$
  2. $序列\{x_n\}发散到负无穷，记为\lim\limits_{n \to +\infty}x_n = -\infty$
  3. $序列\{x_n\}为无穷序列，记为\lim\limits_{n \to +\infty}x_n = \infty$
  4. 非空集合E ⊂ ℝ无上界, 记supE = +∞
  5. 非空集合E ⊂ ℝ无下界, 记supE = −∞

注：不能说{xn}的极限为$+\infty或-\infty$

类收敛序列描述

x为某个有限实数−∞ < x < ∞

无穷大序列的夹挤原理：

对于实数序列{xn}{yn}，假设∃N ∈ ℕ, s.t.∀n > N, 有 − ∞ < xn < yn < +∞，那么

  1. $若\lim\limits_{n \to +\infty}x_n = +\infty,则\lim\limits_{n \to +\infty}y_n = +\infty$
  2. $若\lim\limits_{n \to +\infty}x_n = -\infty，则\lim\limits_{n \to +\infty}y_n = -\infty$

无穷大序列的子序列也是无穷大序列

  1. $若\lim\limits_{n \to +\infty}x_n = +\infty，则\lim\limits_{n \to +\infty}x_{n_k} = +\infty$
  2. $若\lim\limits_{n \to +\infty}x_n = -\infty，则\lim\limits_{n \to +\infty}x_{n_k} = -\infty$
  3. $若\lim\limits_{n \to +\infty}x_n = \infty，则\lim\limits_{n \to +\infty}x_{n_k} = \infty$

实数序列在扩充实数系中至多有一个极限
