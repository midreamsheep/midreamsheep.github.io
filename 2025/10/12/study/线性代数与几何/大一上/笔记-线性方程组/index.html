线性方程

包含x1, x2, ..., xn的线性方程组是形如a1x1 + a2x2 + ... + anxn = b的方程

  1. b与a1, a2, ...an是实数或者复数，通常为已知数
  2. 下标n为任意正整数

线性方程组

线性方程组是由一个或者几个包含相同变量x1, x2, ...xn的线性方程组成

线性方程组的解：一组数(s1, s2, ..., sn)，其分别替代xn时所有方程两边值相等

线性方程组解集：线性方程组所有解的集合叫线性方程组的解集

线性方程组等价：若两个线性方程组由相同的解集，则两者是等价的

线性方程组的解的情况：

  1. 无解
  2. 有唯一解
  3. 有无穷多解

其中2、3成为相容，1成为不相容

初等行变换

基本思路：把方程组用一个更容易解的等价方程组代替

初等行变换：

  1. 倍加变换：把某一行换成它本身与另一行的倍数的和
  2. 倍乘变换：把某一行的所有元素乘以一个非零数
  3. 对换变换：把两行交换

注意：初等行变换是可逆的（显然）

初等行变换不改变解集

证明：

假设一个线性方程组经过初等行变换得到一个新的线性方程组，则易得原方程的一个解是新方程的一个解，又由于初等行变换是可逆的，故新方程的一个解是原方程的一个解

因此：若两个线性方程组是行等价的，则它们具有相同的解集

线性方程组的矩阵

阶梯型矩阵

首元（先导元）：矩阵中最左边的非零元素

阶梯形矩阵（REF）定义：

  1. 每一非零行都在每一零行之上
  2. 某一行的先导元素所在列位于前一列先导元素的右边
  3. 某一先导元素所在列下方的元素都是0

简化阶梯形矩阵（RREF）定义：

  1. 满足阶梯型
  2. 每一非零行的先导元素是1
  3. 每一先导元素1是该元素所在列的唯一非零元素

主元位置：矩阵中的主元位置是A中对应于它的简化阶梯形中先导元素1的位置。

主元列：主元列是A的含有主元位置的列。

主元：主元位置上的非零元素

主元位置在为变换的矩形中不一定是主元（可能为0）

定理-简化阶梯型矩阵的唯一性

每个矩阵行等价于唯一的简化阶梯型矩阵，同时简化阶梯形也是唯一的

求解

定义：

  1. 向前步骤：将矩阵转化为阶梯形矩阵
  2. 向后步骤：将阶梯形矩阵转化为简化阶梯形矩阵
  3. 基本变量：对应于主元列的变量
  4. 自由变量：非基本变量
  5. 解线性方程组：确定无解或者求出参数表达式

空集无参数化秒速

规范化求解：

  1. 转化为简化阶梯形矩阵
  2. 判定是否相容(看增广矩阵最右列是不是主元列)
  3. 找基本变量与自由变量
  4. 求出特解
  5. 求出通解

解的情况

线性方程组相容 ⇔ 增广矩阵最右侧不是主元列 ⇔ 不存在[0...0b]的形式

线性方程组有唯一解 ⇔ 无自由变量

线性方程组有无穷多解 ⇔ 至少有一个自由变量

线性方程组的解集

齐次线性方程组

定义：Ax = 0则称为齐次线性方程组

平凡解：每一个齐次线性方程组都有一个x=0的平凡解

非平凡解：不为0的解

齐次线性方程组有非平凡解 ⇔ 方程至少有一个自由变量

证明：

若有非平凡解，则至少有两个解，即有无穷多个解，故至少有一个自由变量

若至少有一个自由变量，则存在无穷多个解，故有非平凡解

齐次与不齐次解的关系

设方程Ax=b对某个b是相容的，p为一个特解，则Ax=b的解集是所有形如w=p+v的向量的集，其中v是Ax=0的任意一个解

上述即若Ax=b有解，则其解集为Ax=0的解集平移向量p得到，p是Ax=b的任意一个特解

向量方程

视角转换

视角：从行->列 向量方程的形式： 即x1v1 + x2v2 + ... + xpvp = b

又即判断b是否在span{v1, v2, ..., vp}中

向量

定义

列向量（简称向量）：仅含有一列的矩阵

标量乘法（数乘）：cu

向量加法：平行四边形法则

向量减法：u-v = u+(-1)v

代数性质

  1. u+v=v+u
  2. (u+v)+w = u+(v+w)
  3. u+0 = 0+u = u
  4. u+(-u) = -u+u = 0(-u = (-1)u)
  5. c(u+v) = cu + cv
  6. (c+d)u = cu+du
  7. c(du) = (cd)u
  8. 1u = u

线性相关性

线性组合

定义：

给定Rn中的向量v1, v2, ..., vp和标量c1, c2, ..., cp，向量y = c1v1 + c2v2 + ... + cpvp

则称y为向量v1, v2, ..., vp以c1, c2, ..., cp的线性组合

线性无关

对Rn中一组向量{v1, v2, ..., vp}

若x1v1 + x2v2 + ... + xpvp = 0仅有平凡解，则线性无关

若不存在不全为零的权c1, c2, ..., cp, 使上诉方程成立，则称为线性相关

相关特征

令Rn中的S = {v1, v2, ..., vp}

S线性相关 ⇔ S中至少有一个向量是其他向量的线性组合

若S线性相关且v1 ≠ 0, 则存在某个vj(j > 1), 它是前面向量的线性组合

若p > n，则S一定线性相关，证明：Ax = 0中至少有一个自由变量即有非平凡解

若S中有零向量，则一定线性相关

集合的生成

若v1, v2, ..., vp是Rn中的向量，则Span{v1, v2, ..., vp}表示这些向量所有线性组合的向量的集合

矩阵方程

定义

若A是m × n矩阵，它的各列为a1, a2, ..., an, 若x是Rn中的向量

则A与x的积就是A的各列乘以x中对应元素的线性组合

即： 解的存在性：方程Ax = b有解 ⇔ b是A的各列的线性组合

计算规则：若乘积Ax有定义，则Ax的第i个元素是A的第i行元素与x的响应元素乘积之和

性质

  1. A(u+v) = Au + Av
  2. A(cu) = c(Au)
