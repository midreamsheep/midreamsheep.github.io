线性变换

若变换T称为线性的，则：

  1. T(u + v) = T(u) + T(v)
  2. T(cu) = cT(u)

推论：

T(cu + dv) = cT(u) + dT(v)

T(0) = 0

线性变换的矩阵

线性变换

Rn → Rm的线性变换本质上都是x → Ax的矩阵变换, 即T(x) = Ax

证明：

对任意Rn中的x = x1e1 + x2e2 + ... + xnen

其中A称为线性变换T的标准矩阵

满射（映上）

映射T : Rn → Rm称为满射 ⇔ Rm中的任一b是Rn中的像

对A，满射与以下条件同理：

  1. 主元数等于行数
  2. 每一行都有主元
  3. 对任一b，Ax=b有解
  4. A的列生成Rm

单射（一对一）

映射T : Rn → Rm称为单射 ⇔ Rm中的b是Rn中的至多一个x的像

对A，单射与以下条件同理：

  1. 主元数等于列数
  2. 无自由变量、
  3. Ax=0只有平凡解
  4. A的各列线性无关

矩阵运算

和与标量乘法

  1. A+B=B+A
  2. (A+B)+C=A+(B+C)
  3. A+0=A
  4. r(A+B) = rA+rB
  5. (r+s)A = rA+sA
  6. (rs)A = r(sA)

乘法

证明：

A(Bx)，将两次符合映射为(AB)x = Cx

不同的乘法视角

  1. 列视角：AB的每一列都是A的各列以B的对应列的元素为权的线性组合

  2. 行视角：AB的每一行为B的各行以A的对应行元素为权的线性组合

  3. 行列法则：AB的第i行第j列的元素是A的第i行与B的第j列的对应元素乘积之和

    A决定行，B决定列

  4. 列行展开：AB = col1(A)row1(B) + col2(A)row2(B) + ... + coln(A)rown(B)

乘法性质

一般情况下AB ≠ BA

若AB=AC,则一般情况下B=C不成立

若AB是零矩阵，则不能断定A=0或B=0

A为m × n的矩阵, B为n × p的矩阵，AB为m × p的矩阵

矩阵的乘幂

Ak = A × A × ... × A

A0 = I

结合律

A(BC) = (AB)C

左分配律

A(B + C) = AB + AC

显然

右分配律

(B + C)A = BA + CA

显然

数乘

r(AB) = (rA)B = A(rB), r为任意数

矩阵乘法恒等式

InA = A = AIn

转置

AT的列是A的对应行组成

性质

(AT)T = A

(A + B)T = AT + BT

(rA)T = rAT

(AB)T = BTAT

矩阵的逆

定义：

一个n × n的矩阵A是可逆的，若存在一个C使得AC = CA = In

可逆矩阵称为非奇异矩阵，不可逆矩阵称为奇异矩阵

逆的唯一性

假设A存在两个逆B，C

则B = BIn = BAC = InC = C

与前提矛盾，得证

性质

若A, B都是可逆n × n矩阵

(1)Ax = b，对任意b ∈ Rn，存在x使其成立

证明：令x = Bb, 则Ax = ABb = Ib = b, 成立

(2)A−1也可逆且(A−1)−1 = A

(3)(AB)−1 = B−1A−1

推广：若干个n × n可逆矩阵的积可逆且其等于这些矩阵逆相反顺序的乘积

(4)AT也可逆且(AT)−1 = (A−1)T

逆的推导

初等矩阵

易得：初等行变换都能用初等矩阵的左乘表示

注：观察角度为AB中的各行为B各行以A的对应行的值为权的加权和

例如： R3 = (−4)R1 + R3 R2 ↔︎ R1 R3 = 5R3

易得：初等矩阵是可逆的，因为行变换是可逆的

可逆则行等于I

设A是可逆矩阵，则对任意b，方程Ax=b有解，A在每一行有一个主元位置，则行等价于In

推导

若A ∼ In, 则A经过一系列初等行变换可得到In, 则存在初等矩阵Ep，...，E2，E1

则A ∼ E1A ∼ E2E1A ∼ ... ∼ Ep...E2E1A = In

Ep...E2E1A = In

易得，也存在其逆，使得A = (E1−1E2−1...Ep−1)(Ep...E2E1)A = (E1−1E2−1...Ep−1)In = (E1−1E2−1...Ep−1)

因此：

  1. A = (E1−1E2−1...Ep−1)
  2. A−1 = Ep...E2E1
