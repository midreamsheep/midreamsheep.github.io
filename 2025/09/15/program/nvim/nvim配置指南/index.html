neovim介绍

Neovim is a project that seeks to aggressively refactor Vim in order to:

  • Simplify maintenance and encourage contributions
  • Split the work between multiple developers
  • Enable advanced UIs without modifications to the core
  • Maximize extensibility

See the Introduction wiki page and Roadmap for more information.

Neovim 是一个旨在对 Vim 进行激进重构的项目，其目的是：

  • 简化维护工作并鼓励贡献
  • 在多个开发者之间分配工作
  • 无需修改核心即可启用高级用户界面
  • 最大限度地提高可扩展性

请参阅 介绍 维基页面和 路线图 以获取更多信息。

neovim的基本使用

neovim的模式

  • 命令模式（Normal Mode）：用于导航、编辑和执行命令（也称一般模式）。
  • 插入模式（Insert Mode）：用于输入和编辑文本（也称编辑模式）。
  • 可视模式（Visual Mode）：用于选择和操作文本块（也称底行模式）。

常用编辑命令

按键 介绍
d 删除 (Delete)，并进入寄存器。dd 删除整行。可搭配光标移动使用，例如d10h
c 修改 (Change)，删除并进入插入模式。cc 修改整行。同d
y 复制 (Yank)。yy 复制整行。同d
p 粘贴 (Put) 到光标后或下一行。
P 粘贴到光标前或上一行。
r<字符> 替换当前光标字符为指定字符 (Replace)
R 进入替换模式，连续替换多个字符
u 撤销 (Undo)
Ctrl + r 重做 (Redo)，即撤销的撤销
i 在光标前进入插入模式 (Insert)
o 在当前行下方新建一行并进入插入模式 (Open new line)
O 在当前行上方新建一行并进入插入模式
~ 切换当前字符的大小写

标签页操作

快捷键 描述
gt 或 :tabn 切换到下一个标签页 (Next)
gT 或 :tabp 切换到上一个标签页 (Previous)
<n>gt 切换到第 n 个标签页 (e.g., 2gt)

其他操作

快捷键 描述
. 重复上一次的修改操作（极其强大）
Ctrl + a 将光标下的数字增加
Ctrl + x 将光标下的数字减少
>> 向右缩进当前行
<< 向左缩进当前行
== 自动缩进当前行

可视模式

快捷键 描述
v 进入字符可视模式
V 进入行可视模式
Ctrl + v 进入块可视模式（列选择）
gv 重新选择上一次选中的区域
在可视模式下，所有移动命令都有效，用于调整选区范围。选中后，可以用编辑操作 (d, y, c, :, ! 等) 对选区进行操作。
o 在可视模式下，将光标在选区的两端切换

命令模式

文件命令

命令 描述
:e[dit] {file} 打开指定文件 (Edit)。如果文件未修改，会切换缓冲区。
:ene[w] 新建一个未命名的缓冲区 (Edit New)。
:w[rite] 保存文件。
:w[rite] {file} 另存为指定文件。
:w[rite]! 强制写入（覆盖只读文件等）。
:sav[eas] {file} 另存为指定文件，并切换缓冲区到新文件。
:q[uit] 关闭当前窗口（如果缓冲区有修改会警告）。
:q[uit]! 强制关闭当前窗口，丢弃所有修改。
:wq / :x / ZZ 保存并退出当前窗口。
:wq! 强制保存并退出。
:qa[ll] 退出所有窗口 (Quit All)。
:qa[ll]! 强制退出所有窗口，丢弃所有修改。
:wqa[ll] / :xa[ll] 保存所有修改过的缓冲区并退出。

标签页操作

命令 描述
:tabe[dit] {file} 在新标签页中打开文件 (Tab Edit)。
:tabnew {file} 同上，在新标签页中打开文件。
:tabc[lose] 关闭当前标签页。
:tabo[nly] 关闭除当前标签页外的所有其他标签页。
:tabn[ext] 切换到下一个标签页 (等同于 gt)。
:tabp[revious] 切换到上一个标签页 (等同于 gT)。
:tabm[ove] [+N] [-N] 移动当前标签页到第 N 位。:tabm +1 后移一位。

搜索和替换

命令 描述
:/{pattern} 向前搜索 {pattern}。
:?{pattern} 向后搜索 {pattern}。
:noh[lsearch] 暂时关闭搜索高亮，直到下一次搜索。
:s/old/new/ 将当前行第一个 old 替换为 new。
:s/old/new/g 将当前行所有 old 替换为 new (g 表示全局)。
:%s/old/new/g 将全文的 old 替换为 new。
:%s/old/new/gc 全文替换，但每次替换前请求确认。
:n1,n2s/old/new/g 在 n1 到 n2 行之间进行替换。e.g., :10,20s/foo/bar/g

外部命令

命令 描述
:!{cmd} 执行外部 shell 命令，e.g., :!ls -la
:r[ead] {file} 将文件内容插入到光标下方。
:r !{cmd} 将外部命令的执行结果插入到光标下方，e.g., :r !date
:sh 启动一个 shell 子进程，输入 exit 返回 Vim。
:te[rm] 在当前窗口打开一个终端缓冲区。
:mkexrc[!] {file} 将当前所有设置保存为配置文件。
:help {topic} 打开内置帮助文档，最重要的命令！ e.g., :help :w
