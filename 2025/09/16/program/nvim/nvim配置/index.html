选项设置 vim.opt

用途：控制 Neovim 的外观和行为基础设置。

要查看所有可用选项，可以在 Neovim 中运行 :help option-list 或 :options 命令。

📋 显示与界面 (Display & UI)

选项 类型 默认值 描述
number boolean false 显示行号
relativenumber boolean false 显示相对行号
cursorline boolean false 高亮当前行
cursorcolumn boolean false 高亮当前列
signcolumn string "auto" 标记列显示方式 ("yes", "no", "auto", "number")
showmode boolean true 显示当前模式
showcmd boolean true 显示未完成的命令
ruler boolean true 显示光标位置
laststatus number 1 状态行显示 (0-不显示, 1-多窗口时, 2-总是)
statusline string "" 状态行格式
title boolean false 设置终端标题
wrap boolean true 自动换行
linebreak boolean false 在单词边界换行
showbreak string "" 换行行首显示字符
list boolean false 显示不可见字符
listchars table {} 不可见字符的显示方式

⌨️ 编辑行为 (Editing Behavior)

选项 类型 默认值 描述
mouse string "" 鼠标支持 ("a"-所有模式, "n"-普通模式等)
clipboard string "" 剪贴板集成 ("unnamed", "unnamedplus")
undofile boolean false 持久化撤销历史
swapfile boolean true 创建交换文件
backup boolean true 创建备份文件
writebackup boolean true 写入时创建备份
autoread boolean false 文件外部修改时自动重读
autowrite boolean false 自动写入
hidden boolean false 允许隐藏缓冲区
confirm boolean false 操作前确认

按键映射 vim.keymap

基本语法

参数 描述 示例
{mode} 映射生效的模式 'n', 'i', 'v', 'x', 't', ''
{lhs} 左手边 - 要映射的按键 '<leader>w', 'jk'
{rhs} 右手边 - 映射到的命令或函数 '<cmd>w<cr>', function() ... end
{opts} 可选参数表 { silent = true, desc = 'Save file' }

🎯 模式 (Modes)

vim.keymap.set() 支持所有 Vim 模式：

模式字符串 描述 常用场景
'n' Normal 模式 导航、文件操作
'i' Insert 模式 快速退出、特殊输入
'v' Visual 模式 文本选择、操作
'x' Visual 模式 (同 'v')
's' Select 模式
't' Terminal 模式 终端操作
'c' Command-line 模式 命令行快捷方式
'o' Operator-pending 模式 操作符等待
'' 所有模式 通用映射

映射快捷键定义

🎯 核心概念：Leader 键

什么是 Leader 键？

<leader> 是一个前缀键，它本身不执行任何操作，而是用来创建命名空间，避免快捷键冲突。你可以把它想象成编程中的命名空间或计算机上的 Ctrl/Alt 键。

如何设置 Leader 键？

为什么使用 Leader 键？

  • 避免冲突：不与内置快捷键冲突
  • 易于记忆：<leader>w 保存，<leader>q 退出
  • 可扩展性：可以创建大量自定义快捷键
