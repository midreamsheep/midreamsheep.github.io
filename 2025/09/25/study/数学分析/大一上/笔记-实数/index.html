Peano公理

内容

正整数集N定义：

  1. 存在一个元素“0”

  2. 在N上定义一个后继映射

  3. 后继是一一对应的，即

  4. 若S ⊂ N且满足1)0 ∈ S; 2)x ∈ S ⇒ σ(x) ∈ S总成立，则S = N

  5. 将σ(0)定义为1

引理

1.整数集中的每个非0数，都是某个数的后继

∀n ∈ N, 若n ≠ 0, 则∃m ∈ N, s.t.n = σ(m)

证明：

设S = {n ∈ N|n ≠ 0且∃m ∈ N, s.t.n = σ(m)} ∪ {0} 1)0 ∈ S显然成立 2)假设n ∈ S, 要证k = σ(n) ∈ S, 则取n = m, 即有k = σ(m) ∈ S

整数加法定义

1)∀n ∈ N, n + 0 = n 2)∀n, m ∈ N, n + σ(m) = σ(n + m)

交换律

x + y = y + x

即证S = {y ∈ N|x + y = y + x对∀x ∈ N成立} = N

消去律

x + y = z + y，则x = z

定义

  1. 戴德金分割
  2. 区间套理论

无尽小数

定义

形如±a0.a1a2a3....an....为无尽小数

规定±a0.a1a2a3...an999999... = ±a0.a1a2a3....(an + 1),前者为非规范小数

实数=规范小数，定义为ℝ

大小定义

对a = +a0.a1a2a3....an...与b0.b1b2b3....bn....

  1. a > b, 若∃n ∈ N, s.t.ai = bi, 对∀i = 1, 2, ..., n且an > bn
  2. a=b，a<b类似于1
  3. a,b为负数时通过-a,-b获得a，b大小关系

实数有序性

  1. ∀x, y ∈ ℚ x<y,x>y,x=y三种情况有且仅有一个成立
  2. x < y, y < z → x < z

定理

稠密性

∀a, b ∈ ℝ, 那么∃有尽小数c, s.t.a < c < b

证明:

  1. 若a < 0 < b, 则取c = 0

确界原理

上界定义： 实数子集E ⊂ ℝ, E ≠ ∅，若∃p ∈ ℝ, s.t.∀x ∈ E, x ≤ p，则称跑为E的上界

确界原理：

  1. 实数的非空子集有上界则必有上确界
  2. 实数的非空子集有下界则必有下确界
  3. 注：有理数集的情况下不一定成立

下确界的构造方法：通过构造法依次构造a0, a1, ...，使得当an + 1时就不为下确界

证明：

  1. 证明通过构造方法一定为构造出规范小数

    证明若构造出非规范小数，则9循环前一个数能为an + 1，与构造方法矛盾

  2. 证明构造出的规范小数是下确界

    假定构造出的数不是下确界引出一对矛盾

  3. 证明构造出的规范小数是下确界

    假设有个更大的c，然后证明其不是下界
