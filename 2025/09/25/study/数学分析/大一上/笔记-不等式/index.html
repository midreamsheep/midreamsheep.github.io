不等式

三角不等式

|a1 + a2 + ... + an| ≤ |a1|+|a2|+... + |an|

证明（数学归纳法）：

(1)n = 1时，显然成立

n = 2时，分类讨论能够证明(a1, a2的各种大于小于0情况)

(2)假设n时成立，即 : |a1 + ... + an| ≤ |a1|+... + |an|

则对n + 1 : |a1 + ... + an + an + 1| ≤ |a1 + ... + an|+|an + 1| ≤ |aq|+... + |an+1|

伯努利不等式

(x + 1)n ≥ 1 + nx

证明n = 1, 0成立，n ≥ 2时直接二项式展开
